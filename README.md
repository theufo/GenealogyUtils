<h1>ContextMenuTxtCreator</h1>
Добавляет контекстное меню при нажатии правой кнопкой мыши по папке, чтобы быстро создавать *.txt файл с содержимым из буфера обмена

<h1>SpbArchivesLinkSaver</h1>
Расширение для браузера, которое позволяет копировать данные из результатов поиска на сайте spbarchives.ru в формате "%ссылка% %год% %названием дела%"
